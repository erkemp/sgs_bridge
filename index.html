Welcome to sgs_bridge

This repository will be used to help merge and track ideas, suggestions and general information pertaining to the sgs_bridge project.

Overview

This project was started in order to create some level of continuity between Steam Group Chats and the r/SGS community.

Chat rooms are a fantastic resource for any community, and access to live/active members should not be restricted to Steam -- or having to keep the chat window the entire time.

Main Objectives

  • Provide a near-seamless experience between IRC and Steam Group Chat
